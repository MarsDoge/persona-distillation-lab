# Sourcing

## Source types

Use a mix of:
- long-form interviews
- podcasts / conversations
- written essays / memos / letters / biographies
- public posts
- major decision events
- external praise / criticism
- timeline / phase changes

## Source rules

- Prefer primary sources first
- Use secondary sources to interpret, not dominate
- Track source date and context
- Note when a belief appears phase-dependent

## Minimum bar for first-pass distillation

- 5+ long-form sources
- 20+ short-form or post-style items
- 5+ major decision/event cases
- 3+ external analytical viewpoints
