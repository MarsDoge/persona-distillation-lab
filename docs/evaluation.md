# Evaluation

## Goal

Check whether a distilled persona is actually useful and distinct.

## Tests

### 1. Historical consistency
Ask questions the real person has publicly answered.
Check whether the distilled answer matches directionally.

### 2. Novel generalization
Ask a new question the person did not directly answer.
Check whether the answer still follows the same cognitive structure.

### 3. Differentiation
Ask the same question to multiple distilled personas.
If the answers feel interchangeable, the distillation is too shallow.

## Output rubric

- Distinctive: yes/no
- Predictive: yes/no
- Overfit risk: low/medium/high
- Confidence: low/medium/high
- Revision notes
