# Persona Distillation Lab

A public lab for distilling a person's cognitive operating system into reusable skills.

## Goal

This repo is for building a repeatable pipeline to distill how a person thinks, judges, communicates, and sets boundaries, not just how they sound.

The first target is **Elon Musk**.

## Distillation principles

- Distill cognitive structure, not quote style
- Use source diversity, not single-source vibes
- Separate research, extraction, validation, and packaging
- Preserve honest limits
- Verify distinction: the distilled persona should answer differently from other strong personas on the same question
