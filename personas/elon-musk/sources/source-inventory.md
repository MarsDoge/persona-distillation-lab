Status: stage-complete first-pass inventory

## How to use this file
This inventory is for coverage and auditability. Not every source should become a final pattern. A pattern only graduates when it shows stability across domains or decisions.

Why it matters:

### 2. Lex Fridman podcast episode(s)

- secondary synthesis, not primary ground truth

## E. Phase-awareness

The distillation must not assume a frozen Musk. At minimum, compare patterns across:
- early startup / engineering-builder phase
- Tesla / SpaceX scale-up phase
- Twitter / xAI / infrastructure-control phase

Questions:
- which traits are stable?
- which are intensified by power and scale?
- which are phase-specific rather than core?

## F. Promotion standard

Promote a pattern into the final skill only if it satisfies most of these:
- appears across multiple domains
- helps predict response to a new problem
- distinguishes Musk from Jobs / Munger / Naval / generic founder-speak
- can be tied back to public evidence categories
