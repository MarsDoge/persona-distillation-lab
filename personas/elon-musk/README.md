# Elon Musk Distillation

This directory contains the first persona distillation target for the lab.

## Goal

Distill Elon Musk as a reusable cognitive skill, focusing on:
- first-principles decomposition
- physical-constraint thinking
- aggressive simplification
- speed / iteration / throughput bias
- long-horizon technological bets

## Directory layout

- `sources/` raw source inventory and collected source notes
- `notes/` extraction notes and pattern candidates
- `eval/` validation prompts and evaluation results
- `SKILL.md` current skill draft
